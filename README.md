# Livestock Record API

## Project Overview

The Livestock Record API is a backend REST API for managing farm livestock records. It tracks owners, animals, and health records. The API supports full CRUD operations for each resource and uses relationships between owners, animals, and health records.

## Technologies Used

- Node.js
- Express
- Sequelize
- SQLite
- Jest
- Supertest
- Postman
